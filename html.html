프로젝트 진행 요약

Project Summary
기획 Planning
  • 프로젝트 주제 확정: AI 심리상담 및 멀티모달 감정분석 서비스
  • 요구사항 명세: 사용자 시나리오 기반 기능 리스트 및 정책 수립
  • WBS 작성: 전체 개발 일정 산정 및 파트별 R&R 분배
AI Research
  • 모델 선정: 비용 효율적인 무료 오픈소스 모델(Lightweight LLM) 조사
  • 데이터셋 분석: CACTUS(CBT 대화) 및 감정 데이터셋 구조 파악
  • 환경 구축: GPU 서버 세팅 및 PyTorch 의존성 확인
BE Backend
  • 시스템 설계: 전체 서버 아키텍처 및 API 통신 프로토콜 설계
  • DB 모델링: 사용자 및 상담 로그 저장을 위한 ERD(Schema) 구축
FE Frontend
  • 페이지 설계: 메인/상담/분석 등 핵심 페이지 구조(IA) 정립
  • UI 디자인: 디자인 시스템 정립 및 컴포넌트 설계
  • 뷰 구현: 기술 스택 선정 및 기본 뷰 페이지 퍼블리싱 완료

향후 개발 로드맵

Artificial Intelligence

AI

Roadmap
1월 : 멀티모달 학습 및 모델 검증
  • LLM 파인튜닝: CACTUS(CBT기반) 데이터셋 학습
  • 영상 분석: 웹캠을 통한 실시간 표정 분석 테스트
  • 음성 분석: 감정 오판율 개선을 위한 모델 보완
  • 로직 정립: 얼굴·음성·텍스트 통합 감정 추출 확립
  • 감정 대응: 단일/복합 감정에 대한 처리 로직 마련
2월 : LoRA 활용 및 최적화
  • LoRA 적용: 추출된 감정을 LLM에 파인튜닝
  • 대응책 마련: 감정 상태별 LLM 구체적 대응 전략 수립
  • 최종 최적화: 경량화 모델 성능 및 추론 속도 개선
